# Podnośnik trapezowy z napędem śrubowym

Projekt mechaniczny podnośnika trapezowego.
Projekt obejmuje pełny proces opracowania konstrukcji:

- modelowanie 3D części w **Siemens NX**
- wykonanie **rysunków technicznych**
- opracowanie **złożenia**
- przeprowadzenie **obliczeń wytrzymałościowych**
- dobór elementów znormalizowanych

Celem projektu było zaprojektowanie **specjalistycznego podnośnika trapezowego z napędem śrubowym**, przeznaczonego do ręcznego podnoszenia obciążeń.

---

# Wizualizacja złożenia

Projekt przedstawia mechanizm podnośnika trapezowego napędzanego śrubą trapezową.

Mechanizm działa poprzez obrót śruby za pomocą klucza nasadowego, co powoduje zmianę kąta między ramionami i podnoszenie lub opuszczanie konstrukcji.

Główne elementy układu:

- głowica
- stopa
- ramiona węższe
- ramiona szersze
- śruba napędowa
- nakrętka
- tuleja prowadząca
- łożysko wzdłużne
- sworznie

Projekt umożliwia **pełny demontaż konstrukcji oraz wymianę zużytych elementów**.

---

# Parametry konstrukcyjne

| Parametr | Wartość |
|--------|--------|
| Udźwig nominalny | **5100 N** |
| Maksymalna wysokość podnoszenia | **396 mm** |
| Wysokość przy obciążeniu nominalnym | **228 mm** |
| Odległość przyłożenia siły | **24 mm** |
| Współczynnik bezpieczeństwa | **1.7** |

Analiza statyczna została przeprowadzona dla najbardziej niekorzystnego przypadku pracy mechanizmu.

# Modelowanie CAD

Modele części zostały wykonane w programie **Siemens NX**.

Proces projektowy obejmował:

- modelowanie bryłowe wszystkich części
- budowę kompletnego złożenia
- weryfikację współpracy elementów
- przygotowanie dokumentacji technicznej

---

# Dokumentacja techniczna

Dla zaprojektowanych elementów wykonano **rysunki wykonawcze zgodne z zasadami rysunku technicznego**.

Dokumentacja zawiera:

- wymiary geometryczne
- tolerancje
- chropowatości powierzchni
- oznaczenia materiałowe
- przekroje techniczne

Przykładowe rysunki w projekcie:

- rysunek śruby napędowej
- rysunek tulei
- rysunek złożeniowy podnośnika

# Analiza wytrzymałościowa

W ramach projektu wykonano pełną analizę wytrzymałościową elementów konstrukcyjnych.

Sprawdzeniu poddano między innymi:

### Śruba napędowa

- naprężenia zredukowane
- warunek samohamowności gwintu
- naciski powierzchniowe

### Nakrętka

- zginanie
- ścinanie
- naciski powierzchniowe
- koncentrację naprężeń

### Ramiona

- zginanie
- ściskanie mimośrodowe
- stateczność
- wyboczenie

### Sworznie

- zginanie
- ścinanie
- naciski powierzchniowe

- nakrętka

- głowica

Użyta w tulejach zabezpieczających.
